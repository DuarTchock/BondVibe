# BondVibe - Project Charter

**Version**: 1.0  
**Date**: November 12, 2025

## 1. Project Vision

BondVibe es una plataforma de conexión social que facilita reuniones grupales presenciales auténticas mediante matching basado en personalidad.

**Mission**: Transformar la soledad urbana en conexiones significativas.

## 2. Problem Statement

- Epidemia de soledad en ciudades
- Apps de citas fallan en crear comunidad (enfoque 1:1)
- Meetup carece de matching por personalidad
- Airbnb Experiences muy costoso ($100+)
- Demanda post-pandemia de conexiones IRL

## 3. Solution

### Core Innovation
Matching basado en Big Five + experiencias lideradas por expertos

### Key Features MVP1
1. Test de personalidad (10 preguntas, 4 idiomas)
2. 16 arquetipos de personalidad
3. Eventos curados por expertos
4. Sistema de seguridad (verificación, reportes)
5. Event Buddy System
6. Icebreakers gamificados

## 4. Target Audience

- Singles buscando amistad (no dating)
- Parejas expandiendo círculo social
- Nuevos residentes en ciudad
- Introvertidos (grupos pequeños estructurados)
- Expertos monetizando habilidades

## 5. Success Metrics (Year 1)

- 70% tasa de usuarios repetidos (3 meses)
- NPS >70
- $96K revenue
- 5,000 descargas
- 500+ eventos ejecutados

## 6. Tech Stack

- Frontend: Expo (React Native)
- Backend: Firebase (<$5/mes)
- Database: Firestore
- Auth: Firebase Auth
- Payments: Stripe
- Email: SendGrid

## 7. Roadmap

### MVP1 (Weeks 1-12): Foundation
- User auth + profiles
- Personality test
- Event browsing/booking
- Payment integration
- Host portal
- Safety features

### MVP2 (Weeks 13-24): Enhancement  
- Gamified icebreakers
- Event Buddy System
- Virtual events
- Community pages
- Expand to Tulum

### MVP3 (Weeks 25-36): Scale
- Multi-city expansion
- Advanced recommendation engine
- Subscription tiers
- White-label for communities

## 8. Pilot Strategy

**Launch City**: Guadalajara, Mexico

**Why**: Mid-size (5M metro), tech hub, lower competition, local knowledge

**Pilot Metrics (3 months)**:
- 500 registered users
- 50 events executed
- 20 verified hosts
- 5 venue partnerships
- NPS >60

## 9. Business Model

**Revenue Streams**:
1. Event commissions (15-20%)
2. Premium subscriptions ($9.99/mo)
3. Venue partnerships

**Pricing**:
- Free: Browse events, basic matching, 2 events/month
- Premium: Unlimited events, priority booking, exclusive experiences
- Events: $15-50 per person
