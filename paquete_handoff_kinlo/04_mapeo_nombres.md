# Kinlo · Mapeo de nombres (rebrand BondVibe → Kinlo)

Marca / producto: **Kinlo**. Nombre interno anterior: **BondVibe**.

## Importante para Claude Code
- El **repositorio / proyecto** puede seguir llamándose `bondvibe` (nombre interno). **No es obligatorio** renombrarlo para que la feature funcione.
- La carpeta de recursos `design_handoff_bondvibe_theme/` **conserva su nombre** a propósito, para no romper rutas de código existentes. Su contenido (tokens, componentes) sigue siendo válido.
- Regla: identificadores en **minúsculas** `bondvibe` en código pueden mantenerse; **todo lo visible al usuario** (UI, copys, textos) debe decir **Kinlo**.

## Tiers / naming oficial
- App / producto: **Kinlo**
- Suscripción del anfitrión: **Kinlo Pro** (incluye Community Matching)
- Suscripción del asistente: **Kinlo Plus** (matches ilimitados)
