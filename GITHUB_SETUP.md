## Daily Workflow

### Working on a feature
```bash
# Create branch
git checkout -b feature/BV-XXX-description

# Make changes and commit

# Push to GitHub

```

### Sync changes
```bash
# Pull latest
git pull origin main

# Push your work
git push origin main
```

```bash
# See status

# See history
git log --oneline

# See branches
git branch -a
```
